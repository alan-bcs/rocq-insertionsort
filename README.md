## Estrutura

* `insertion_sort.v`: Arquivo contendo todo o código fonte, definições e provas formais.
* `relatorio.pdf`: Relatório detalhado sobre as estratégias de prova.

## Como Compilar

O projeto foi desenvolvido para ser compatível com **Coq 8.12** (devido ao ambiente jsCoq utilizado). 
Para verificar as provas, recomenda-se o uso do [jsCoq Scratchpad](https://jscoq.github.io/scratchpad.html) ou uma instalação local compatível.
